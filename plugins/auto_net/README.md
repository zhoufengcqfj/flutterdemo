# autoNetFlutter 框架
提供 auto_net:gen，auto_net:genHelp等可执行命令。在需要生成网络协议的目录中，
执行 flutter pub run auto_net:gen命令即可生成网络bean和网络Api。
执行 flutter pub run build_runner build
